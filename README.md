xpdf python
===============================

version number: 0.0.1
author: Edward Atkins

Overview
--------

Python wrapper for xpdf

Installation / Usage
--------------------

To install use pip:

    $ pip install xpdf_python


Or clone the repo:

    $ git clone https://github.com/ecatkins/xpdf_python.git
    $ python setup.py install
